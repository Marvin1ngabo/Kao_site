# KASO - Kigali Autism Support Organization

A modern web application for the Kigali Autism Support Organization, dedicated to supporting individuals with autism and their families in Rwanda.

## About KASO

KASO is committed to providing comprehensive support, education, and advocacy for individuals with autism spectrum disorder and their families in Kigali and throughout Rwanda.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Clone the repository
git clone https://github.com/Marvin1ngabo/Kao_site.git

# Navigate to the project directory
cd Kao_site

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm test` - Run tests
